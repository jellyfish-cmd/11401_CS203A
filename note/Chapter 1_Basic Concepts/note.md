# 什麼是演算法？（What is an Algorithm）
An algorithm is a finite set of instructions that, if followed, accomplishes a particular task.
### 演算法是一組有限步驟，只要照做，就一定能把事情完成。
###   關鍵不是「聰明」，
###   而是 「一定能照做、一定會停下來」。

# 任何演算法，都一定要滿足 5 個條件。
## 1.Input（輸入）
There are zero or more quantities that are externally supplied.
### 演算法可以有輸入，也可以沒有，但「你要不要吃飯」要說清楚。
## 2.Output（輸出）
At least one quantity is produced.
### 一定要有結果，不然你在忙什麼？
## 3.Definiteness（明確性）
Each instruction is clear and unambiguous.
### 每一步都不能「看心情」
    錯誤例子:
    「把數字整理一下」
    「看起來差不多就好」
    正確例子:
    「比較第 i 個與第 j 個」
    「如果 A < B，交換」
## 4.Finiteness（有限性）
The algorithm terminates after a finite number of steps.
### 作業系統那種「一直跑」的東西，在理論上不算演算法。
## 5.Effectiveness（可執行性）
Every instruction must be basic enough to be carried out by a person using pencil and paper.
### 就算很慢，但做得到，才算數。

# Algorithm vs Program
Algorithm（演算法）：一定會結束

Program（程式）：可以一直跑（例如作業系統）
